Mетоды для перебора массива
for: Это классический цикл, который позволяет перебирать элементы массива по одному, используя индекс.
Цикл:
JavaScript:
    
const arr = ["a", "b", "c"];
for (let i = 0; i < arr.length; i++) {
console.log(arr[i]);
}

for...of: Это более современный цикл, который позволяет перебирать элементы массива без использования индекса.
Цикл:
JavaScript:
    
const arr = ["a", "b", "c"];
for (const element of arr) {
console.log(element);
}
    
Цикл for...of в JavaScript используется для перебора элементов итерируемых объектов, таких как массивы, строки, наборы (sets), карты (maps) и другие. Вот краткое описание синтаксиса и пример использования:
forEach: Этот метод позволяет перебирать элементы массива и выполнять для каждого элемента заданную функцию.
Цикл:
JavaScript:
    
const arr = ["a", "b", "c"];
arr.forEach((element) => {
console.log(element);
});

map: Этот метод позволяет преобразовать каждый элемент массива в новый элемент, используя заданную функцию.
Функциональный метод:
JavaScript:
    
const arr = [1, 2, 3];

const doubledArr = arr.map((element) => element * 2);
        
console.log(doubledArr); // [2, 4, 6]

filter: Этот метод позволяет отфильтровать элементы массива, оставляя только те, для которых заданная функция возвращает true.
Функциональный метод:
JavaScript:
    
const arr = [1, 2, 3, 4, 5];

const evenArr = arr.filter((element) => element % 2 === 0);
        
console.log(evenArr); // [2, 4]

reduce: Этот метод позволяет свести массив к одному значению, используя заданную функцию.
Функциональный метод:
JavaScript:
    
const arr = [1, 2, 3, 4, 5];

const sum = arr.reduce((accumulator, currentValue) => accumulator + currentValue);
        
console.log(sum); // 15

some: Этот метод проверяет, существует ли хотя бы один элемент массива, для которого заданная функция возвращает true.
Функциональный метод:
JavaScript:
    
const numbers = [1, 2, 3, 4, 5];

// Проверяет, есть ли хотя бы одно четное число
const hasEven = numbers.some((number) => number % 2 === 0);
        
console.log(hasEven); // true
        
// Проверяет, есть ли хотя бы одно число больше 10
const hasGreaterThanTen = numbers.some((number) => number > 10);
        
console.log(hasGreaterThanTen); // false

every: Этот метод проверяет, для всех ли элементов массива заданная функция возвращает true.
Функциональный метод:
JavaScript:
    
const numbers = [2, 4, 6, 8];

// Проверяет, все ли элементы четные
const allEven = numbers.every((number) => number % 2 === 0);
        
console.log(allEven); // true
        
const numbers2 = [2, 4, 6, 9];
        
// Проверяет, все ли элементы больше 5
const allGreaterThanFive = numbers2.every((number) => number > 5);
        
console.log(allGreaterThanFive); // false
